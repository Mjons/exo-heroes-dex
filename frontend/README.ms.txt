# NFT Collection Organizer

This web-based application allows users to organize and view their NFT collection. Users can filter NFTs by traits, view enlarged images, and change background colors of PNG images.

## Project Structure

The project is divided into two main parts:

1. Frontend: A React application
2. Backend: An Express.js server

## Prerequisites

- Node.js (v14 or later)
- npm (v6 or later)
